Welcome to the Blog!

This blog is ready for your first post. To get started, create markdown files in the _posts directory with the following naming convention:

YYYY-MM-DD-post-title.md

Example Post Structure:

---
layout: post
title: "Your First Blog Post"
date: 2024-01-15 10:00:00 -0000
tags: [research, technology, thoughts]
author: Your Name
excerpt: "A brief description of your post that will appear in the blog list."
---

# Your Post Content Here

Write your post content using Markdown syntax. You can include:

- **Bold text** and *italic text*
- [Links](https://example.com)
- Images: ![Alt text](path/to/image.jpg)
- Code blocks
- Lists and more!

## Subheadings

Your blog supports all standard Markdown features and will automatically 
generate a beautiful, readable post with the site's clean styling.

Tips for Great Blog Posts:

  • Use descriptive tags to help readers find related content
  • Write engaging excerpts that summarize your post
  • Include relevant images to break up text
  • Use headings to structure your content
  • Keep paragraphs concise for better readability
